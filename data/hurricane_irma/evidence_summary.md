# Evidence Summary — Hurricane Irma (NOAA 2021 Report, AL112017)

This document summarizes directly mentioned cause–effect relations between hazards and impacts in the NOAA National Hurricane Center Tropical Cyclone Report: Hurricane Irma (AL112017).

## Source Span Notation

- **AL112017_Irma.pdf** — the NOAA Irma report file  
- **p.xx** — the page number (where known)
